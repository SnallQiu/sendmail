'''vertion1.2'''
新增config设置，默认设置均在config里设置，运行前请先进行config配置


‘’‘模块’‘’

邮件模块：基于SMTP

代理ip切换模块：用了socket和httplib2下的socks模块 后经测试发现有些python版本也可以直接引用socks模块



‘’‘运行’‘’

直接用python跑起来xundailitest.py 文件

只有一个输入，即输入发件人起始位置，发件人文本文件在sendemail根目录的fromadd2.txt下面 ，可以自行添加，但是要注意脚本里的发信人提取规则，只提取密码为字母开头的发信人

脚本为无限循环发送，每天23.30自动停止，该时间可以自由设置 在timem函数里面

第二天继续发送时候查看前一天发送到的发信人位置，第二天接着该位置来



‘’‘逻辑’‘’

一个发信箱在一个ip下面发送最多440封邮件 ，然后登录下一个发信箱继续发送

如果代理ip连接失误，继续换ip，并且依然用该发信箱发送。



‘’‘常见问题’‘’

Q：脚本在跑，红色邮件界面也一直在跑，但是就是发送失败，发送量很小？

A：一，该问题一个是因为发送文本文件问题，文本被识别为垃圾文件，导致发不过去。解决方法：定期更换text文本。

   二，是这段时间代理质量低，解决方法：等待高质量ip的到来



Q：脚本跑起来无线错误，仔细一看错误显示有ipv6 donsn‘t support

A：该问题为快代理下的白名单问题，解决方法：登录快代理账号，把当前主机ip设置为快代理白名单后即可解决。
